# lfg

> 一个Vue项目，wower玩家发起互动与招募，以及一些游戏经验的Web应用。

> 在线地址：http://54.83.146.225

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
